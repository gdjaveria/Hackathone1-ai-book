This guide provides instructions on how to run the Docusaurus project and view the new homepage.

## Prerequisites

- Node.js >= 20.0
- npm (or yarn)

## Running the Project

1. **Navigate to the book directory**:
   ```bash
   cd physical-ai-book
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm start
   ```

4. **View the homepage**:
   Open your web browser and navigate to `http://localhost:3000`. You should see the new homepage for the "Physical AI & Humanoid Robotics" book.
